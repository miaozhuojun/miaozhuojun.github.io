### 进展

- AArch64寄存器

  - r29(FP)寄存器用于保存调用者的栈帧起始地址

  - r30(LR)寄存器用于保存函数跳转指令(bl)下一条指令的地址

- AArch64的stack限制

  - 进程只能访问stack的空间是一个闭区间：[SP, stack-base - 1]

  - 如果进程通过SP来访问内存，则SP mod 16 = 0，也就是必须四字对齐

- BETA_OPERATION的传参过程

  - arg0(double) -> x0

  - arg1(double) -> x1

  - arg2(long) -> x2

  - arg3(double) -> d0

  - arg4(void \*) -> x3

  - arg5(long) -> x4

  - arg6(void \*) -> x5

  - arg7(long) -> x6

  - arg8(double \*) -> x7

  - arg9(long) -> [sp]
