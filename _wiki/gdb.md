## 打印变量

- 打印十六进制数

  ```shell
  (gdb) p/x val
  ```

## 设置断点

- `b func_name`
- `b file_name:line_number`
- `b *address`

## 修改变量

```shell
(gdb) print x=<num>
```

## 单步调试

通常情况下，step 命令和 next 命令的功能相同，都是单步执行程序。不同之处在于，当 step 命令所执行的代码行中包含函数时，会进入该函数内部，并在函数第一行代码处停止执行。

查看汇编，并单步：

```shell
display /10i $pc
si/ni
```
